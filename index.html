What Makes a Circle?

Wikipedia

“It is the set of all points in a plane that are at a given distance from a given point, the centre; equivalently it is the curve traced out by a point that moves so that its distance from a given point is constant.”

Distance Basics

Or... Crap it's math

Distance Formula

 

$$\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$$

Pro Tips

  • Ditch the $\sqrt{}$, often times we only need to check against $distance^2$
  • Order isn't important, the $()^2$ removes the sign

Distance Check


var radius = 100;     // check radius
var pX, pY, tX, tY;   // player & turret position

var d2 = ((tX - pX) * (tX - pX)) + ((tY - pY) * (tY - pY));
var r2 = radius * radius;

if (d2 < r2) { /* HIT! */ }
if (d2 >= r2) { /* MISS! */ }

Vector Addition

 

Let    $\vec{u} = \langle u_x, u_y \rangle$   and   $\vec{v} = \langle v_x, v_y \rangle$

$\vec{u}+\vec{v} = \langle u_x+v_x, u_y+v_y \rangle$

$\vec{u}-\vec{v} = \vec{u}+(-\vec{v})$

Pro Tips

  • A $\vec{vector}$ is a point with with intention
  • $X = x_1 + x_2$, $Y = y_1 + y_2$

Direction


var pX, pY, tX, tY;   // player & turret position
var bX, bY;           // bullet position

// Get a point relative to the turret
var x = pX - tX;
var y = pY - tY;

// Calculate the angle using atan2
var angle = Math.atan2(y, x);

// Calculate the direction using sin/cos
var x = Math.cos(angle);
var y = Math.sin(angle);

// Multiply the direction by units to move by
bX += x * 5;
bY += y * 5;
 

Motion


var delta;            // delta time in seconds
var pX, pY, tX, tY;   // player & turret position
var bX, bY;           // bullet position
var bT;               // bullet timer (accumulator)

var x = pX - tX;
var y = pY - tY;

var angle = Math.atan2(y, x);

// Shift the angle by the wave produced by sin()
bT += delta;
angle += Math.sin((bT * Math.PI) * 2) * (10 * deg2rad);

var x = Math.cos(angle);
var y = Math.sin(angle);

bX += x * 5;
bY += y * 5;
